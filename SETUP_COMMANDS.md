# Perintah yang perlu dijalankan di terminal

## 1. Jalankan migrasi baru
```bash
php artisan migrate
```

## 2. Jalankan seeder untuk data kriteria (mata kuliah), alternatif (karir), dan bobot
```bash
php artisan db:seed
```

## 3. Clear cache
```bash
php artisan config:clear
php artisan cache:clear
php artisan view:clear
```
